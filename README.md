  ## 주요 기능

- **YOLOv5 객체 감지**  
  실시간 웹캠 영상에서 포트홀을 탐지합니다.

- **포트 감지 시 동작 실행**  
  포트홀이 감지되면 `motion_id=18` 명령을 로봇에 전송합니다.

- **기본 자세 복귀**  
  포트홀이 일정 시간 동안 감지되지 않으면 3초 후 `motion_id=0`으로 복귀합니다.

- **포트 선택 GUI**  
  사용자에게 시리얼 포트를 선택하도록 안내합니다.

- **멀티스레딩 처리**  
  감지를 별도의 스레드에서 처리해 GUI가 멈추지 않습니다
